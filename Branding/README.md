# OpenSight — Branding & Identidade Visual

> **Codinome do sistema visual:** `WARMIND CARMESIM`
> Uma fusão entre a estética da *Warmind* **Rasputin** (Destiny) e o **gavião-real / harpia brasileira** (*Harpia harpyja*).

Esta pasta concentra **toda** a identidade visual do OpenSight. Antes de criar qualquer interface, slide, documento, ícone ou material de marketing, consulte estes arquivos para manter o padrão estético.

---

## Conceito em uma frase

> **OpenSight é a Warmind que guarda a sua vida financeira — uma harpia de aço que vê tudo, toca em nada.**

A marca traduz visualmente os dois pilares do produto:

| Pilar do produto | Tradução visual |
|------------------|-----------------|
| **Security-first / READ-ONLY** | Estética de guardião militar, monolítica, blindada (Rasputin) |
| **Visão / insight financeiro** | A harpia — predador de topo, visão aguçada (OpenSight) |

## Regra de ouro

A identidade nasce de **três tensões controladas**. Toda peça deve respeitá-las:

1. **Escuro x Brasa** — fundos quase-pretos (`void`) com pontos de carmesim/brasa controlados. O vermelho é poder, não enfeite: use com parcimônia.
2. **Frio (aço) x Quente (âmbar)** — a estrutura é metálica e fria; o detalhe nobre é o âmbar/latão da Warmind.
3. **Geometria x Organismo** — a estética é angular e construtivista, mas o símbolo é vivo: a harpia.
